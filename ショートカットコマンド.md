# Emmet Wrap with Abbreviation のショートカット設定
Emmet Wrap with Abbreviation は、指定行をHTMLタグで囲むこと

## 設定場所

VS Code の **keybindings.json** に書きます。

---

## 手順（いちばん簡単）

1. VS Code を開く  
2. `Ctrl + Shift + P` を押す  
3. 「Keyboard Shortcuts (JSON)」と入力して選択  

以下を入力

```json
[
  {
    "key": "ctrl+shift+w",
    "command": "editor.emmet.action.wrapWithAbbreviation",
    "when": "editorTextFocus"
  }
]
